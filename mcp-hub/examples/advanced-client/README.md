# Advanced Client

高级 MCP Client 示例，展示服务缓存、负载均衡、并行调用。

## 功能

| 特性 | 说明 |
|------|------|
| 服务缓存 | 缓存服务发现结果，减少 Hub 查询 |
| 自动路由 | 根据工具名自动选择服务 |
| 并行调用 | 同时调用多个服务 |
| 代理对比 | 代理调用 vs 直接连接性能对比 |

## 运行

```bash
# 先启动 Hub 和所有服务
# 然后运行
uv sync
uv run python client.py --hub http://localhost:8000
```

## 演示内容

1. **服务发现** - 按标签、工具名发现服务
2. **自动路由** - 调用 calculate 自动路由到 calc-server
3. **并行调用** - 同时获取天气、时间、计算结果
4. **性能对比** - 代理调用 vs 直接连接
